# UK Business Validator 🏢

Check if a UK company is real. Search by name or company number and get back the company status, incorporation date, registered address, and officer details.

Uses the [Companies House](https://find-and-update.company-information.service.gov.uk/) public search — no API key required.

## What It Does

- Search by company name or registration number
- Shows company status (active, dissolved, etc.)
- Displays incorporation date
- Shows registered office address
- Links directly to the Companies House listing

## How to Use

1. Open `index.html` in your browser
2. Type a company name or number
3. Click Search

That's it. No server, no API key.

## Why This Exists

Before working with a new client or supplier, it's worth checking they're a real, active company. This tool makes that quick and easy.

## Tech

- Plain HTML, CSS, and JavaScript
- Companies House public web search
- No dependencies
